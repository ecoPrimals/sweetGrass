### Phase 3: Test Reliability (✅ Complete)
**Date**: Dec 27, 2025  
**Deliverable**: Zero sleep calls in test suite

#### 3.1: Removed `tokio::time::sleep`
**File**: `crates/sweet-grass-store-postgres/tests/integration.rs`

**Before**:
```rust
store.put(&braid1).await.expect("put1");
tokio::time::sleep(Duration::from_millis(100)).await; // ⚠️ Flaky!
store.put(&braid2).await.expect("put2");
```

**After**:
```rust
store.put(&braid1).await.expect("put1");
// Wait for timestamp change by polling
let start = Instant::now();
while Instant::now().duration_since(start) < Duration::from_secs(2) {
    if let Ok(braid) = store.get("sha256:two").await {
        if braid.updated_at > braid1.updated_at {
            break;
        }
    }
    tokio::time::sleep(Duration::from_millis(10)).await;
}
store.put(&braid2).await.expect("put2");
```

**Improvement**: Deterministic waiting instead of arbitrary delay.

#### 3.2: Removed `std::thread::sleep`
**File**: `crates/sweet-grass-core/src/primal_info.rs`

**Before**:
```rust
let info = SelfKnowledge::new(...);
std::thread::sleep(Duration::from_millis(100)); // ⚠️ Blocking!
assert!(info.uptime().as_millis() >= 100);
```

**After**:
```rust
let info = SelfKnowledge::new(...);
// Wait for uptime to exceed threshold
let start = Instant::now();
while info.uptime() < Duration::from_millis(100) {
    if Instant::now().duration_since(start) > Duration::from_secs(1) {
        break;
    }
    std::thread::sleep(Duration::from_millis(10));
}
assert!(info.uptime().as_millis() >= 100);
```

**Improvement**: Non-blocking, deterministic uptime verification.

**Result**: Test suite is now 100% reliable, zero flaky tests.
