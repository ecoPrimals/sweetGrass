## 🎯 QUICK REFERENCE

### Common Tasks

**Build Release**:
```bash
cargo build --release
```

**Run Tests**:
```bash
cargo test --lib                    # Unit tests
cargo test -- --ignored             # Integration tests (requires Docker)
```

**Start Service**:
```bash
./target/release/sweet-grass-service --port 8091
```

**Check Health**:
```bash
curl http://localhost:8091/health
```
